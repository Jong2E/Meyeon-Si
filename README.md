# 커서 테스트 프로젝트

이 저장소는 다양한 웹 개발 프로젝트들을 포함하고 있습니다.

## 프로젝트 구성

### 1. 미연시 (Visual Novel)
- 위치: `미연시/`
- 설명: 비주얼 노벨 스타일의 웹 애플리케이션

### 2. 커리큘럼 장바구니 (Curriculum Cart)
- 위치: `커리큘럼 장바구니/`
- 설명: 커리큘럼 관리 웹 애플리케이션

### 3. Prompt 프로젝트
- 위치: `prompt/`
- 설명: 프롬프트 관련 웹 애플리케이션

### 4. 테스트 222
- 위치: `테스트 222/`
- 설명: 테스트 프로젝트

## 시작하기

각 프로젝트 폴더로 이동하여 `index.html` 파일을 웹 브라우저에서 열어보세요.

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Google Apps Script (Code.gs)

## 라이선스

이 프로젝트는 개인 학습 목적으로 제작되었습니다.
